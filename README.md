## Installation

1. Clone the repository.

```sh
git clone https://github.com/oguzhankaymak/React-Native-Unit-Test.git
```

2. Install expo.

```sh
npm install --global expo-cli
```

3. Install necessary packages via yarn.

```sh
yarn install
```

5. Run the app.

```sh
expo start
```

5. Run the tests.

```sh
yarn run test --watch
```
